## 📋 Project Overview
This is a solo developer project focused exclusively on exploring and testing the **sales and transaction data endpoints** of the Etsy Open API v3. The primary goal is to understand Etsy's OAuth 2.0 authentication flow and best practices for secure, compliant access to shop sales data.

This application is for **personal development and testing purposes only**. It will never be distributed or made available to any third parties.

---

## 🎯 Core Objectives
- Implement and test Etsy's OAuth 2.0 authorization flow for seller accounts
- Validate order and transaction data retrieval endpoints
- Understand API rate limiting and error handling patterns
- Develop a basic framework for personal sales data analysis

## ✅ Compliance Commitment
This project strictly adheres to all Etsy API Terms of Use and developer guidelines:

- 🔒 **Only connects shops owned or explicitly authorized by the developer**
- 🔑 **Never collects or stores Etsy user passwords**
- 🚫 **No web scraping or bypassing of Etsy's official interfaces**
- 📧 **No unsolicited messages or marketing communications**
- 🤝 **No selling, sharing or redistribution of Etsy member data**
- 🤖 **No use of Etsy data to train artificial intelligence models**
