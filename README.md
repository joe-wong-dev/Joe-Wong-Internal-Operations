# Joe Wong Internal Operations
Personal learning and testing project for the Etsy Open API v3.

## Project Status
🚧 **Under Development - Private Preview Only** 🚧

This is a solo developer project to explore and test the Etsy Open API. The primary goal is to understand Etsy's OAuth 2.0 authentication flow and basic API endpoints for shop management workflows.

## Current Focus
- Testing Etsy OAuth 2.0 authorization flow
- Reading shop profile and listing data
- Testing order transaction data retrieval
- Understanding API rate limits and best practices

## Planned Features
- Basic order management dashboard
- Inventory status tracking
- Shipping fulfillment workflow testing

## Compliance
This project strictly adheres to all of Etsy's API Terms of Use:
- ✅ Only connects shops owned or explicitly authorized by the developer
- ✅ Never collects or stores Etsy user passwords
- ✅ No web scraping or bypassing of Etsy's official interfaces
- ✅ No unsolicited messages or marketing
- ✅ No selling or sharing of Etsy member data
- ✅ No use of Etsy data to train AI models

## Important Note
This is a private development project. It is not a public SaaS product and is not available for use by any third parties. If this project ever expands beyond personal use, commercial access will be requested from Etsy before any additional users are allowed.
